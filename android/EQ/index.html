Moodle EQ Generator

Generate essay questions to import into Moodle with {wildcard} support.

Ever wondered whether you could create an essay question for Moodle with variables that change value so each student gets a unique quiz?

Now you can, use this tool to generate as many questions with or without wildcards and generate the Moodle XML to import.

Currently only OR wildcards are supported, but range is planned for the future. Start by clicking the "Add Another Question" button.

Wildcard Examples:

  • {Bob|Jill|Alice} wondered what to do...
  • Subnet for {10|13|14} hosts the following base network: 192.168.2.0/24.
  • Discuss why {distance|time|speed} is important.

Each wildcard question will create a category in the format of Question Name-Question Number featuring all possible combinations of wildcards in your question.

Once you have imported your questions into Moodle simply create a Quiz with randomised questions from the category created for each wildcard question.
